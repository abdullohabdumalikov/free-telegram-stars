# ⚠️ Lag Machine Project

Hello!

This project does not provide any practical tools or utilities for everyday use.  
Instead, it is intentionally designed as a learning resource to demonstrate how a "Lag Machine" works.

The purpose of this project is educational — to help you understand:
- how performance issues can be created,
- what causes system or browser lag,
- and how such problems can be analyzed and prevented.

⚡ Use this knowledge responsibly.
